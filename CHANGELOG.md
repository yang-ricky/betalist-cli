## [Unreleased]

### Added
- Initial release of betalist-cli
- Commands: `latest`, `startup`, `markets`, `doctor`, `config`
- HTML scraping as primary data source
- Optional API integration with token authentication
- File-based caching with configurable TTL
- JSON, YAML, and table output formats
- Comprehensive health check with `bl doctor`
- CI/CD with GitHub Actions
- Daily selector and API health monitoring

### Notes
- This is an unofficial tool, not affiliated with BetaList
- Jobs feature is not included in v1
- Search is best-effort only
